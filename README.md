# Bonnet1.github.io

My GitHub Pages site for indie web games and prototypes.

## What's here
- `index.html`: the landing page for my projects and updates.
- `christmas-defense/`: Gingerbread Gauntlet, a tower-defense style holiday game.

## Running locally
Open `index.html` in a browser. To jump straight into the game, open `christmas-defense/index.html`.
